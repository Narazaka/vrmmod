# VRM Mod

VRM avatar mod for Minecraft. Replaces player models with VRM 1.0 avatars.

## Supported Platforms

- Fabric
- NeoForge

## Minecraft Version

- 1.21.4

## Dependencies

### Fabric

- [Fabric API](https://modrinth.com/mod/fabric-api)
- [Fabric Language Kotlin](https://modrinth.com/mod/fabric-language-kotlin)
- [Architectury API](https://modrinth.com/mod/architectury-api)
- [Cloth Config](https://modrinth.com/mod/cloth-config) (for settings screen)
- [Mod Menu](https://modrinth.com/mod/modmenu) (optional, for settings access)

### NeoForge

- [Kotlin for Forge](https://modrinth.com/mod/kotlin-for-forge)
- [Architectury API](https://modrinth.com/mod/architectury-api)
- [Cloth Config](https://modrinth.com/mod/cloth-config) (for settings screen)
